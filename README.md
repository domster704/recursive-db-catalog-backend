# AitiGuru test task

## Начало

Надо выполнить следующие команды по порядку:

1. Установить все зависимости

```bash
poetry install
```

2. Запустить контейнер PostgreSQL и контейнер бэка

```Bash
docker compose up -d --build
```

3. Запустить миграцию в БД через Alembic

```bash
poetry run alembic upgrade head
```

4. Заполнить базу данных начальными данными.<br/>
   Это можно легко автоматизировать, перенеся функцию seed_initial_data в функцию lifespan в `container.py` , но я хотел
   показать, что умею создавать свои программы CLI через Typer

```Bash
poetry run seed-db
```

## Описание решения

- Решение построено на веб-сервере FastAPI с использованием ORM на SQLAlchemy и SQLModel.
- Миграции базы данных совершал через Alembic.
- Архитектура приложения построена по методологии "Чистая слоистая архитектура" с использованием DDD
- Контейнеризация применялась для сервиса FastAPI и СУБД PostgreSQL
- Все SQL коды находятся в папке `sql` (разделение по задачам видно по названиям файлов)
- Посмотреть документацию и попробовать выполнить запрос на FastAPI можно по `http://<ip>:8000/docs`
